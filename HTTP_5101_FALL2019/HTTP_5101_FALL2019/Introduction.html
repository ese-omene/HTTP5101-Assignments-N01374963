This is my first HTML web page!

HTML is a scripting language that allows people to put static information on the internet.

The information on static pages won't change. Some examples of static websites include:

  • Brochure Pages
  • About Pages
  • "Frequently Asked Questions" Pages
  • Pages like the one you're on now!

Static pages are the simplest type of webpages.
